# 用户风控冻结

# 0. 文档概述

本文档描述 ForX 永续合约 **用户风控冻结** 的后台管理能力与业务规则，对应原型如下：

| 端 | 原型路径 |
|---|---|
| 后台 · 冻结配置 | `perp_dex/admin/用户冻结设置.html` |
| 后台 · 冻结日志 | `perp_dex/admin/冻结操作记录.html` |

> 费率相关能力见 `document/费率.md`，两者权限隔离（费率管理员 / 风控管理员）。

---

# 一、业务逻辑

## 1.1 冻结项说明

运营可在后台对用户配置以下冻结项，**可组合叠加**：

| 配置项 | 说明 |
|---|---|
| 提现冻结 | 禁止链上提现 |
| 下单冻结 | 禁止开新仓、加仓、挂单 |
| 仅平仓模式 | 仅允许平已有仓位 |
| 资金划转冻结 | 禁止账户间划转 |
| API 交易冻结 | 禁止 API Key 下单 |
| 账户冻结 | 最高级别，禁止一切资金行为 |

**不包含**：

- **充值冻结**：链上上账无法冻结。
- **登录冻结**：无业务场景，不在后台提供。

## 1.2 风控标签

- 配置抽屉内支持**自定义输入**风控标签。
- 留空时列表显示「正常」。
- 标签用于运营内部识别与分类，可与冻结项独立配置。

## 1.3 生效规则

- 后台保存后**立即生效**。
- 冻结项与风控标签变更均须填写**操作备注**。
- 所有写操作写入**冻结操作记录**，支持事后审计追溯。

# 二、后台管理页面

## 2.1 用户冻结设置

**原型**：`perp_dex/admin/用户冻结设置.html`  
**权限**：风控管理员。

### 列表筛选

| 字段 | 说明 |
|---|---|
| 钱包地址 | 模糊匹配 |
| UID | 模糊匹配 |
| 冻结状态 | 正常 / 部分冻结 / 账户冻结 |

### 列表字段

| 列 | 说明 |
|---|---|
| UID / 钱包 | 用户标识 |
| 风控标签 | 自定义标签，空为「正常」 |
| 冻结状态 | 综合冻结项计算 |
| 冻结项摘要 | 当前生效的冻结配置 |
| 操作 | 配置冻结 |

### 配置抽屉

1. **冻结项开关**：提现 / 下单 / 仅平仓 / 划转 / API / 账户冻结
2. **风控标签**：自定义输入
3. **操作备注**（必填）

### 子页面

- 侧栏入口 → **冻结操作记录**（`冻结操作记录.html`）

---

## 2.2 冻结操作记录

**原型**：`perp_dex/admin/冻结操作记录.html`

### 筛选

| 字段 | 说明 |
|---|---|
| UID / 钱包 | 目标用户 |
| 操作人 | 后台账号 |
| 操作类型 | 配置冻结 / 撤销冻结项 / 变更风控标签等 |
| 时间范围 | 起止日期 |

### 列表字段

| 列 | 说明 |
|---|---|
| 操作时间 | UTC+8 展示 |
| 操作人 | 后台账号 |
| 目标用户 | UID + 钱包 |
| 操作类型 | 配置冻结、变更风控标签等 |
| 变更详情 | 冻结项 diff / 标签变更 |
| 备注 | 操作备注摘要 |

每条日志至少包含：**操作人、操作时间、目标用户（UID / 钱包）、操作类型、变更详情、备注**。

# 附录：名词说明

| 名词 | 说明 |
|---|---|
| 部分冻结 | 存在一项或多项冻结，但未达到账户冻结 |
| 账户冻结 | 最高级别冻结，禁止一切资金行为 |
| 风控标签 | 运营自定义的用户风险标识，不影响链上充值 |
