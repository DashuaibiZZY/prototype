## 三、新币上线倒计时（原型演示）

### 3.1 背景

新上线代币在正式开盘前，尚无 K 线行情与盘口数据。页面需支持两种展示状态：

1. **正常 K 线**：已有行情的合约，展示蜡烛图示例（默认状态）
2. **新币上线倒计时**：未开盘合约，图表区域整页展示距离开盘倒计时

为方便产品/设计评审，在原型中提供**专用切换按钮**，可在两种状态间来回演示（正式上线后移除）。

### 3.2 切换入口

| 项目 | 说明 |
|------|------|
| 位置 | `#view-kline` 工具栏右侧（放大图标旁） |
| 样式 | 橙色虚线边框小按钮，标注「原型演示专用」 |
| 正常模式文案 | `切换：新幣倒計時` |
| 倒计时模式文案 | `切换：正常K線` |
| 元素 ID | 正常模式：`#btn-kline-proto-toggle`；倒计时模式：`#btn-kline-proto-toggle-cd` |
| 触发函数 | `toggleKlineMode()` |

> 两种状态下均保留切换按钮，避免进入倒计时后无法切回。

### 3.3 状态一：正常 K 线（默认）

- 容器：`#kline-state-normal`
- 展示内容：周期切换、工具栏、左侧绘图工具、蜡烛图示例、成交量、时间轴、底部状态栏
- 交易对示例：`BTCUSDT` · 100x
- 市场信息栏、订单簿、最新成交均展示正常模拟数据

### 3.4 状态二：新币上线倒计时

- 容器：`#kline-state-countdown`（默认 `hidden`）
- 示例代币：`PEPEUSDC` · 永续 · 50x 最高杠杆 · USDC 结算

#### 顶部工具栏

- 左侧：`新幣上線` 橙色标签 + `PEPEUSDC·永續`
- 右侧：保留「圖表 / 深度 / 詳情 / 資金費率」Tab 切换（与正常模式一致，不影响其他视图）

#### 主内容区（居中展示）

| 元素 | 内容 |
|------|------|
| 代币图标 | 圆形占位，字母 `P` |
| 标题 | `PEPE / USDC 永續合約` |
| 副标题 | `新幣永續合約即將開啟交易，敬請期待` |
| 倒计时标题 | `距離交易開啟` |
| 倒计时数字 | 天 / 时 / 分 / 秒 四格，等宽 monospace 数字框 |
| 开盘时间 | `2026-07-11 12:00:00 (UTC+8)` |
| 合约信息 | 最高杠杆 50x · 结算币种 USDC · 合约类型 永续 |

#### 倒计时行为（原型）

- 初始值：`02天 14时 35分 22秒`（约 2.6 天）
- 每秒递减，个位数前补零（`padCountdown`）
- 变量：`klineCountdownRemaining`（秒）

#### 底部提示

`交易開啟前暫無行情數據 · 開盤後將自動切換至 K 線圖表`

### 3.5 联动规则：市场信息栏

切换至**新币倒计时**后，`<!-- 2. 市場信息欄 -->` 联动更新：

| 字段 | 正常模式 | 倒计时模式 |
|------|----------|------------|
| 交易对名称 | `BTCUSDT` | `PEPEUSDC` |
| 杠杆标签 | `100x` | `50x` |
| 最新价 | `65,432.10`（绿色） | `--`（灰色） |
| 24h 涨跌幅 | `-1.76%` | `--` |
| 标记价格 | `65,391.3` | `--` |
| 指数价格 | `65,391.3` | `--` |
| 资金费率 / 倒计时 | `0.0001% 00:25:13` | `--` |
| 24h 成交量 | `60.23 M` | `--` |
| 未平仓合约 | `3.23 M` | `--` |
| 24h 最高价 | `66,332.5` | `--` |
| 24h 最低价 | `61,332.5` | `--` |

**规则**：除交易对名称与杠杆标签外，其余字段一律显示 `--`。

相关元素 ID：`#market-pair-name`、`#market-pair-leverage`、`#market-last-price`、`#market-change`、`#market-mark-price`、`#market-index-price`、`#market-funding`、`#market-volume`、`#market-oi`、`#market-high`、`#market-low`

切回正常 K 线后，恢复 `marketNormalSnapshot` 中的原始数据。

### 3.6 联动规则：订单簿

切换至**新币倒计时**后，`<!-- 订单簿容器 -->` 展示空状态：

| 项目 | 行为 |
|------|------|
| 卖单列表 `#sell-orders` | 清空 |
| 买单列表 `#buy-orders` | 清空 |
| 中间最新价 `#ob-last-price` | 显示 `--` |
| 标记价 `#ob-mark-price` | 显示 `--` |
| 空状态遮罩 `#ob-book-empty` | 显示，文案：`暫無掛單數據` |

表头（价格 / 数量 / 累计）与底部精度、资产选择器仍保留，仅数据区为空。

切回正常模式后，调用 `renderOrderBook()` 重新渲染模拟盘口。

### 3.7 联动规则：最新成交

切换至**新币倒计时**后，「最新成交」Tab 展示空状态：

| 项目 | 行为 |
|------|------|
| 成交列表 `#trades-list` | 清空 |
| 空状态遮罩 `#ob-trades-empty` | 显示，文案：`暫無成交記錄` |

在倒计时模式下切换至「最新成交」Tab 时，不调用成交数据渲染；空状态遮罩根据当前 Tab 自动显隐。

切回正常模式后，若当前在最新成交 Tab，调用 `renderTrades()` 恢复模拟数据。

### 3.8 不受影响的区域

以下区域在倒计时切换时**不联动变更**（保持原样）：

- 深度图 / 详情 / 资金费率视图（`switchView` 逻辑不变）
- 右侧交易终端（开仓/平仓、下单表单）
- 下半区委托/持仓列表

### 3.9 技术实现要点

```
toggleKlineMode()
  ├─ 切换 #kline-state-normal / #kline-state-countdown 显隐
  ├─ applyMarketBarState(snapshot)     // 市场信息栏
  ├─ applyOrderBookEmptyState(isEmpty) // 订单簿 + 最新成交
  └─ renderKlineCountdown()            // 进入倒计时时刷新数字

renderOrderBook() / renderTrades()
  └─ 若 isKlineCountdownMode === true，跳过数据渲染
```

## 四、止盈止损下单（交易终端）

### 4.1 入口

交易终端订单类型 Tab 新增/完善「止盈止损」：

- Tab ID：`#tab-profit-loss`
- 模式值：`profit-loss`
- 切换函数：`switchOrderMode('profit-loss')`

### 4.2 三种订单模式互斥

| 模式 | 价格输入区 | 止盈止损区 | GTC 选择器 |
|------|------------|------------|------------|
| 限价委托 `limit` | 显示 | 隐藏 | 显示 |
| 市价委托 `market` | 显示（置灰「市價」） | 隐藏 | 隐藏 |
| 止盈止损 `profit-loss` | 隐藏 | 显示 | 隐藏 |

### 4.3 止盈止损表单

容器：`#tpsl-order-container`（默认隐藏）

#### 委托方向

- 下拉选项：**单向委托** / **双向委托**
- 变量：`currentTPSLMode`
- 切换后重新渲染触发价区域：`renderTPSLTriggers()`

#### 触发价 + 委托价（动态渲染）

根据**开平仓**与**单向/双向**组合，动态生成表单项：

| 场景 | 单向委托 | 双向委托 |
|------|----------|----------|
| 开仓 | 1 组「触发价格」 | 2 组「触发价格」（止盈/止损各一组） |
| 平仓 | 1 组「触发价格」 | 1 组「止盈」+ 1 组「止损」 |

每组包含：

1. **触发价输入** + 价格类型下拉（最新成交价 / 标记价格）
2. **委托价格类型**下拉（市价 / 限价）
3. **委托价输入**（选市价时置灰显示「市价」）

### 4.4 交互细节

- 从市价模式切回止盈止损时，恢复价格输入框可用状态，避免残留置灰
- 选择「市价」委托时，委托价输入框 `disabled`，显示居中「市价」
- 选择「限价」委托时，委托价输入框可编辑
- 切换开平仓（`switchPositionSide`）时，若当前为止盈止损模式，需联动刷新触发区域布局

## 五、原型演示指引

### 演示：正常交易流程

1. 打开 `perp_dex/合约交易.html`
2. 默认即为 BTCUSDT 正常 K 线 + 完整盘口
3. 可在交易终端切换限价 / 市价 / 止盈止损体验下单表单

### 演示：新币上线倒计时

1. 点击 K 线工具栏右侧 **`切换：新幣倒計時`**
2. 观察联动效果：
   - 图表区 → 倒计时页
   - 市场信息栏 → 除交易对外均为 `--`
   - 订单簿 → `暫無掛單數據`
   - 切换「最新成交」Tab → `暫無成交記錄`
3. 点击 **`切换：正常K線`** 恢复全部数据
